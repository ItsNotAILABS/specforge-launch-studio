# Product Surface

SpecForge Launch Studio is a desktop tool for builders and teams who need to draft app specifications quickly, manage work, and prepare clone-ready apps for launch.

## Core Surfaces

1. Home Dashboard: Builder/Launcher split, quality score, next action, clone-flow state.
2. Spec Builder: click-only 10-section export flow.
3. Portfolio: project workspace with detailed table view and filters.
4. Marketplace: clone-ready app listings with creator profiles.
5. Launcher: 6-step post-clone onboarding and pricing advisor.

## 10-Section Export

1. App Type
2. Audience
3. Core Promise
4. Primary Workflow
5. Feature Set
6. Data Model
7. Team Roles
8. Integrations
9. Launch Plan
10. Success Metrics
